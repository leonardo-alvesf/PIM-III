Sintaxe Básica e Tipos de Dados em C#

Estrutura de um Programa C#

using System; namespace MeuPrograma { class Program { static void Main(string[] args) { Console.WriteLine("Olá, Mundo!"); } } }

Tipos de Dados Primitivos:

Numéricos Inteiros:

  • byte: 0 a 255
  • int: -2.147.483.648 a 2.147.483.647
  • long: números muito grandes

Numéricos Decimais:

  • float: 7 dígitos de precisão
  • double: 15–16 dígitos de precisão
  • decimal: 28–29 dígitos (use para dinheiro)

Outros Tipos:

  • bool: true ou false
  • char: caractere único ('A')
  • string: texto ("Olá")

Declaração de Variáveis:

// Declaração explícita int idade = 25; string nome = "João"; bool ativo = true; // Inferência de tipo var preco = 99.90m; // decimal var cidade = "São Paulo"; // string

Operadores:

Aritméticos:

  • + (soma), - (subtração)
  • * (multiplicação), / (divisão)
  • % (resto da divisão)

Comparação:

  • == (igual), != (diferente)
  • >, <, >=, <=

Lógicos:

  • && (E), || (OU), ! (NÃO)

Estruturas de Controle:

// If-Else if (idade >= 18) { Console.WriteLine("Maior de idade"); } else { Console.WriteLine("Menor de idade"); } // Switch switch (diaSemana) { case 1: Console.WriteLine("Segunda"); break; case 2: Console.WriteLine("Terça"); break; default: Console.WriteLine("Outro dia"); break; }

Classes e Objetos

Em C#, uma classe é o molde e o objeto é a instância criada a partir dele.

Definindo uma Classe:

public class Pessoa { // Propriedades public string Nome { get; set; } public int Idade { get; set; } // Construtor public Pessoa(string nome, int idade) { Nome = nome; Idade = idade; } // Método public void Apresentar() { Console.WriteLine($"Olá, sou {Nome}, {Idade} anos."); } }

Criando Objetos:

// Instanciando um objeto Pessoa p1 = new Pessoa("Ana", 30); Pessoa p2 = new Pessoa("Carlos", 25); // Chamando método p1.Apresentar(); // Olá, sou Ana, 30 anos. p2.Apresentar(); // Olá, sou Carlos, 25 anos.

Pilares da OOP:

Encapsulamento

  • Oculta detalhes internos da classe
  • Usa public, private, protected

Herança

  • Uma classe herda membros de outra
  • Usa o operador : — ex.: class Aluno : Pessoa

Polimorfismo

  • Mesmo método, comportamentos diferentes
  • Usa virtual e override

Abstração

  • Expõe apenas o essencial ao mundo externo
  • Classes e métodos abstract
