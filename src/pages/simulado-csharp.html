Qual tipo de dado em C# deve ser usado para armazenar valores monetários?

Escolha uma alternativa

O tipo decimal deve ser usado para valores monetários porque oferece 28-29 dígitos de precisão e evita erros de arredondamento que ocorrem com float e double.

Qual pilar da Orientação a Objetos permite que uma classe herde características de outra?

Escolha uma alternativa

Herança é o pilar da OO que permite que uma classe filha herde propriedades e métodos de uma classe pai, promovendo reuso de código e criando hierarquias de tipos.
